# OhMyFood

Intégration de la maquette OhMyFood

### Prérequis

Node >= 14
NPM

### Installation pour développement

1. Clonez le répo
2. Installez les modules dépendantes: `npm install`
3. Lancez Sass avec PostCSS: `npm run dev`
4. Lancez le serveur local de Live Server
